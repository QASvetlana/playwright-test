# Playwright Test Project for Topklik

Автоматизированные UI тесты для b2b решения Topklik с использованием Playwright и Allure отчетности.

## 📋 Реализованные тест-сценарии

### 1. Успешная авторизация
- Проверка возможности входа в систему

### 2. Переключатель "Скрыть столешницу"
- Проверка, что при включении переключателя столешница не отображается

### 3. Переключение на П-образную столешницу
- Проверка, что при выборе П-образной формы отображается соответствующая столешница

### 4. End-to-End сценарий: Собрать заказ

**Параметры заказа:**
- Форма столешницы: П-образная
- Толщина: 4
- Плинтус: не требуется (выключен)
- Остров: добавлен
- Проточки для стока воды: добавлены
- Цвет: выбран

**Шаги сценария:**
1. Нажать кнопку "Рассчитать"
2. Нажать кнопку "Расчет"
3. Проверить открытие страницы с результатами расчета

**Проверки результатов:**
- ✅ Материал: acryl
- ✅ Тип столешницы: П-образная
- ✅ Опции: Проточки для стока воды
- ✅ Итоговая стоимость (проверка, что цена больше 0)
